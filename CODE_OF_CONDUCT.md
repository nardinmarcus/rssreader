# Code of Conduct

QMReader follows a simple maintainer-led contribution model:

- Be respectful and specific.
- Discuss code, behavior, and evidence rather than personal attacks.
- Do not post private data, API keys, credentials, or third-party confidential content.
- Keep issues and pull requests focused on the project.
- The maintainer may close conversations that are abusive, spammy, off-topic, or unsafe.

For security reports, use [SECURITY.md](SECURITY.md) instead of public threads.
